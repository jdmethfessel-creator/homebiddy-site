# homebiddy-site

Marketing site for HomeBiddy — your home buying buddy.

Single-page Next.js (Pages Router) site. Mobile-first, max-width 480px. Form posts to Formspree (`xpqkeaqe`).

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

Deployed on Vercel at homebiddy.com.
